8.24. getPaymentEntry [_new_]¶

Formát príkazu:

Parameter paymentID:

Identifikátor naprogramovanej platby.

Môže nadobúdať hodnoty v rozsahu paymentID = 1 až NumPayments.

Výstupný parameter paymentID:

Identifikátor naprogramovanej platby je zhodný so vstupným parametrom paymentID.

Výstupný parameter paymentName:

Názov naprogramovanej platby.

Výstupný parameter paymentType:

Typ naprogramovanej platby.

Môže nadobúdať jednu z hodnôt uvedených v parametri paymentType príkazu setPaymentEntry.

Návratové hodnoty:

Chyba

Popis

EFP_BAD_PAYMENT

Parameter paymentID bol poslaný nesprávne.

Príkaz zabezpečí vrátenie údajov o naprogramovanej platbe.
